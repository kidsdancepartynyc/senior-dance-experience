# Senior Dance Experience Website

Static one-page website built for GitHub Pages.

## Files
- `index.html` — website content
- `styles.css` — layout and styling
- `script.js` — navigation and subtle scroll animation
- `assets/` — logo, favicon and founder photos
- `CNAME` — custom domain configuration for GitHub Pages
